# DigitalClock

html/css/js/github pagesの練習

Bootstrap一切使わずに頑張った

### 2021/09/24時点の機能
- デジタル時計
- フォント切り替え
- 一時停止

### 今後追加予定の機能
- ストップウォッチ/タイマー機能
- ダークモード対応
- レスポンシブ対応(PC全画面/スマホ横画面に最適化するつもり)
